K-means

  • Color based K-means segmentation

  • This method segments the image based on a KNN clustering of the colors in the image. The image is represented as lab colors (wikipedia page).
  • Input: image I, number of segments K
Output: labeled image L

colors = []
for row in 1..rows, for col in 1..cols
  colors.append(I(row, col))

clustering = kmeans(colors, k)

for row in 1..rows, for col in 1..cols
  L(row, col) = clustering.clusterLabelOf(I(row, col))
