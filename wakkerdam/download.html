Installation and Download

First download this file: Wakkerdam.py.

We are assuming you already installed python (version 2.x). On ubuntu, install some required libaries like this:

sudo apt-get install libgv-python
sudo apt-get install python-pygraphviz

These packages are not installed on the wingtip systems. In general the libgv package is installed by installing graphviz (http://www.graphviz.org/) with python support. If not installed, the system will work when the "import gv" is removed and graph drawing functionalities are not used.

Python 3 does not support the graphviz library we use.

Run like this:

python Wakkerdam.py

Or: 

python2.x Wakkerdam.py
